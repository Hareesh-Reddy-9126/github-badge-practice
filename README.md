## How to Earn the Galaxy Brain Badge

The Galaxy Brain achievement is unlocked when your answer in a GitHub Discussion is marked as the accepted answer.

Steps to earn it:
1. Go to the [GitHub Community Discussions](https://github.com/orgs/community/discussions)
2. Browse questions in categories like General, Programming Help, or Actions
3. Find a question you can help with and post a helpful, detailed answer
4. If the question author marks your answer as accepted, you earn progress toward the Galaxy Brain badge
